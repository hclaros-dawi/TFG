# TFG: Plataforma de Ejercicio y Nutrición

## Objetivos

La **Plataforma de Ejercicio y Nutrición** tiene como objetivo proporcionar una solución integral para la gestión del ejercicio y la nutrición. Los usuarios podrán obtener recomendaciones personalizadas para mejorar su salud y alcanzar sus objetivos de bienestar.

### Características principales:
- **Interactuar con el Usuario**: Herramientas interactivas para calcular IMC (Índice de Masa Corporal), calorías y nutrientes recomendados según el perfil del usuario.
- **Recetas Personalizadas**: En función de los productos comprados, se sugieren recetas fáciles y rápidas para mantener una alimentación saludable.
- **Canasta de Compras Inteligente**: Optimización de compras que sugiere productos que se ajusten a los objetivos nutricionales del usuario (por ejemplo, alto en proteínas, bajo en azúcar, etc.).
- **Motivación**: Contenido educativo que incluye vlogs, consejos sobre suplementación, rutinas de ejercicio y más.
- **Personalización Dinámica**: El sistema proporciona recomendaciones personalizadas basadas en los datos ingresados por el usuario, ajustándose a sus objetivos específicos de nutrición y ejercicio.

## Tecnologías Utilizadas

- **Back-End**: PHP (Laravel)
- **Front-End**: HTML5, CSS3, JavaScript
- **Base de Datos**: MySQL
- **Control de Versiones**: GitHub

## Descripción del Proyecto

Este proyecto tiene como objetivo ayudar a los usuarios a alcanzar sus metas de salud y bienestar a través de una plataforma en línea donde podrán calcular su IMC, recibir recomendaciones de dieta, ver recetas adaptadas a sus necesidades, y optimizar su canasta de compras para lograr sus objetivos nutricionales de forma eficiente.
